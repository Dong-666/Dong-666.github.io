Windows项目部署（Node）

安装相关node服务器所需包(art-template模板引擎以及express框架)

搭建一个简单的node服务器

修改下npm run server时的打包配置，因为node服务器中相关的资源文件放在public文件夹内，所以我们定义打包后的资源引用路径（vue.config.js）

之后执行打包

将css、js、img文件夹复制到public文件夹下

将html以及图标文件复制到view下

node服务器根目录下运行服务

这样就部署完成，其实这个和我们平时在开发环境中执行的npm run server过程类似
