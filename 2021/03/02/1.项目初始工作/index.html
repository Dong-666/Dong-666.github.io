划分目录结构

src下的目录结构

  • network （网络模块）→相关网络请求（axios）
  • components （组件）
    • common （公共模块）→可以多项目复用
    • content （业务模块）→该项目可以复用
  • views （路由页面模块）
  • common （公共js文件–常量…）
  • assets （静态资源）
  • router （路由）→页面跳转
  • store （vuex状态管理）
