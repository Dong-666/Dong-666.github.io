|阅读量:

简介

nrm(npm registry manager )是npm的镜像源管理工具，可以使用这个工具快速地在不同的npm源间切换

安装

全局安装

常用命令

  • nrm ls

    查看可以选择的npm源

    带*的是当前使用的源

  • nrm use 源

    这个是用来切换源的

    例如：nrm use taobao就是使用淘宝镜像源

  • nrm del 源

    删除某个源

  • nrm test 源

    测试某个源的速度

  • nrm add 源名 源路径

    添加其它源
